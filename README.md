# Game Planning

Central planning and issue tracking for the game project.

## Repos

| Repo | Description | Tech Stack |
|------|-------------|------------|
| `deployments` | Smart contracts & deployment scripts | Hardhat/Solidity |
| `web3-backend` | Web3 backend services | Hardhat |
| `game` | Game client | Unity/C# |
| `frontend` | Web frontend | React/TypeScript |

## Issue Labels

### Area Labels
- `area:deployments` - Smart contracts
- `area:backend` - Web3 backend
- `area:game` - Unity game
- `area:frontend` - Web frontend

### Type Labels
- `type:feature` - New feature
- `type:bug` - Bug fix
- `type:tech-debt` - Technical debt

### Priority Labels
- `priority:high` - Do first
- `priority:medium` - Do soon
- `priority:low` - Do eventually

## Usage

```bash
cd game-planning
/plan-day --hours 6
```
